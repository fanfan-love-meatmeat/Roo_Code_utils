## 1. 当前调用方式现状

### 1.1 两种现有入口

| 入口 | 方式 | 是否需改 server.py | 使用场景 |
|------|------|:---:|------|
| **MCP 工具** | AI Agent 通过 MCP stdio 调用 `patent_md_to_docx` | ❌ 已注册，无需每次改 | AI Agent 自动化工作流 |

### 1.2 缺失的入口：CLI

**当前不存在命令行直接调用方式**。用户若想从终端转换一个 MD，必须写一个临时 Python 脚本：

```python
# 临时脚本 —— 每次都要手写
import sys; sys.path.insert(0, '...')
from tools.patent_md_to_docx import build_patent_docx
with open('input.md') as f: doc = build_patent_docx(f.read())
doc.save('output.docx')
```

这不是工具固化，这是手工作坊。

## 2. CLI 化方案

### 2.1 目标形态

```bash
# 最简调用
python -m tools.patent_md_to_docx path/to/input.md

# 指定输出路径
python -m tools.patent_md_to_docx path/to/input.md -o path/to/output.docx

# 自动推断输出目录
# 输入 test/patent_md/xxx.md → 输出 test/patent_docx/xxx_{timestamp}.docx
```

### 2.2 与现有代码的关系

```
tools/patent_md_to_docx.py (现有)
  ├── build_patent_docx(md_text) → Document   [核心引擎，已有]
  └── if __name__ == '__main__':               [CLI 入口，待加]
        cli_main()
```

CLI 入口仅需约 15 行代码，跟在 `build_patent_docx` 之后。**不修改任何现有函数签名，不改变 server.py**。server.py 和 CLI 共享同一核心引擎 `build_patent_docx`。

### 2.3 CLI 参数设计

| 参数 | 必需 | 说明 |
|------|:---:|------|
| `md_path` (位置参数) | ✅ | Markdown 文件路径 |
| `-o, --output` | ❌ | 输出 DOCX 路径。默认: MD 上浮一层 `patent_docx/` 下 `{MD名}_{时间戳}.docx` |
| `--no-timestamp` | ❌ | 禁用时间戳后缀，直接覆盖写入 |

### 2.4 实现骨架

```python
# tools/patent_md_to_docx.py 末尾追加

if __name__ == '__main__':
    import argparse, os, datetime

    parser = argparse.ArgumentParser(
        description='专利 MD → CNIPA 合规 DOCX 转换器'
    )
    parser.add_argument('md_path', help='Markdown 文件路径')
    parser.add_argument('-o', '--output', help='输出 DOCX 路径（可选）')
    parser.add_argument('--no-timestamp', action='store_true',
                        help='禁用时间戳后缀')
    args = parser.parse_args()

    md_path = os.path.abspath(args.md_path)
    if not os.path.isfile(md_path):
        parser.error(f'文件不存在: {md_path}')

    with open(md_path, 'r', encoding='utf-8') as f:
        doc = build_patent_docx(f.read())

    if args.output:
        out = args.output
    else:
        fn = os.path.splitext(os.path.basename(md_path))[0]
        ts = '' if args.no_timestamp else f'_{datetime.datetime.now().strftime("%Y%m%d_%H%M")}'
        out_dir = os.path.join(os.path.dirname(os.path.dirname(md_path)), 'patent_docx')
        os.makedirs(out_dir, exist_ok=True)
        out = os.path.join(out_dir, f'{fn}{ts}.docx')

    doc.save(out)
    print(out)
```

三者共享单一核心引擎，互不耦合。
