## 2. 架构安全自检

### 2.1 防御机制实装清单

| 防御点 | 位置 | 策略 |
|--------|------|------|
| 字体回退阻断 | `patent_style_utils.py:set_run_font()` — `w:hint='eastAsia'` | 阻断 Word 等线回退 |
| 空白 strong 过滤 | `patent_style_utils.py:add_rich_paragraph()` — `not txt.strip()` | 消除 bold spaces 冗余 Run |
| 页边距合规 | `patent_style_utils.py:init_patent_document()` — Cm(2.5)/Cm(1.5) | CNIPA 官方标准 |
| 表格降级 | `patent_md_to_docx.py:_parse_table_block()` — 空 rows_data 提前返回 | 畸形表格不阻塞主流程 |
| ordered_list 跨段合并 | `patent_md_to_docx.py:_parse_ordered_list_block()` — `'; '.join()` | claim 内多 paragraph 不丢内容 |
| 发明名称限定 | `patent_md_to_docx.py:build_patent_docx()` — `prev_was_h1_description` 仅 `说明书` | 其他 H1 后不会误判 |
| MCP 输入防御 | `server.py:_validate_input_file()` | 路径合法性 + 扩展名白名单 + 文件大小上限 |
| 异常分类 | `server.py:patent_md_to_docx()` — FileNotFoundError / PermissionError 分别处理 | AI Agent 可基于错误信息自我纠错 |
