## 5. 环境差异适配记录

| 差异 | 原因 | 处理 |
|------|------|------|
| `from fastmcp.exceptions import ToolError` → 导入失败 | 当前 conda 环境使用 `mcp.server.fastmcp` 内置版，`ToolError` 位于 `mcp.server.fastmcp.exceptions` | 修正为 `from mcp.server.fastmcp.exceptions import ToolError` |
| `mask_error_details=True` → 参数不支持 | 内置版 FastMCP __init__ 不包含此参数（独立 `fastmcp` 包 v2.4+ 支持） | 移除参数，添加注释说明升级路径 |
| 命令行直接调用 `tools/docx_to_md.py` → ModuleNotFoundError | `tools` 包不在 sys.path | 设置 `PYTHONPATH` 环境变量解决（MCP 服务启动时由 server.py L4 自动处理） |
