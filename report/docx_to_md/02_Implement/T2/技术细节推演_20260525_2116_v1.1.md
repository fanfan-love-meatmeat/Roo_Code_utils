## 2. 作用域隔离策略

### 2.1 双语锚点定位

在 Markdown 输出中搜索以下锚点（仅在参考文献区域生效）：

```python
REF_ANCHORS = [
    r'^# 参考文献\b',          # 一级标题：中文
    r'^## 参考文献\b',         # 二级标题：中文
    r'^# References\b',        # 一级标题：英文
    r'^## References\b',       # 二级标题：英文
    r'^# Bibliography\b',      # 一级标题：备用
    r'^## Bibliography\b',     # 二级标题：备用
]
```

### 2.2 文档切割 + 区间清洗

```python
def _normalize_reference_section(md_content: str) -> str:
    """
    1. 双语锚点定位参考文献区域
    2. 切割文档 → 仅对锚点之后区域执行 [N] 换行清洗
    3. 拼接回正文（正文 0 污染）
    """
    for anchor_pattern in REF_ANCHORS:
        m = re.search(anchor_pattern, md_content, re.MULTILINE)
        if m:
            cut_point = m.start()
            body = md_content[:cut_point]       # 正文部分 → 不修改
            ref_section = md_content[cut_point:] # 参考文献区域 → 清洗

            # 仅对锚点后区域执行引文标识换行
            ref_section = re.sub(
                r'(?<!\n)(\[\d+\])',            # 前方非换行的 [数字]
                r'\n\1',                         # 前方插入换行
                ref_section
            )
            return body + ref_section
    
    return md_content  # 未找到锚点 → 原样返回
```

### 2.3 验证：正文不污染

| 场景 | 正文示例 | 清洗结果 |
|---|---|---|
| 行内引用 | `在 Agri-CPS [12] 中` | **不变** (在锚点之前) |
| References 区内 | `[1] Author... [2] Author...` | `\n[1] Author... \n[2] Author...` |
