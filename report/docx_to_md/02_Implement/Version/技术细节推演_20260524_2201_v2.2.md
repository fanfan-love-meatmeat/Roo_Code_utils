## 1. 四项缺陷逐一验证

### 🔴 缺陷 1：align 多行公式内容被 `.*?` 全部删除 — **确认致命 Bug**

原代码：
```python
s = re.sub(r'\\begin\{align\*?\}.*?\\end\{align\*?\}', '', s, flags=re.DOTALL)
```
`.*?` 会匹配 `\begin{align}` 到 `\end{align}` 之间的所有内容，导致多行矩阵、联立方程等核心数学表达式被整块删除，最终输出空字符串。

**修复**：与 equation 处理方式对齐，只剥离环境标签，保留内部公式。

---

### 🔴 缺陷 2：多线程 PyTorch 模型实例化导致 VRAM OOM — **确认风险**

**检索结论**：
- PyTorch 模型在 `eval()` + `torch.no_grad()` 下对 GPU 推理是线程安全的（CUDA 操作期间释放 GIL）
- pix2tex 的 ViT+Transformer 模型约 500MB 显存
- 若每个线程独立实例化 `LatexOCR()`，`max_workers=4` 将消耗 2GB+ 显存，极易 OOM
- 正确做法：全局懒加载单例 + 所有线程共享

**修复**：
```python
_ocr_model = None
_ocr_lock = threading.Lock()

def _get_ocr_model():
    global _ocr_model
    if _ocr_model is None:
        with _ocr_lock:
            if _ocr_model is None:  # 双重检查锁定
                from pix2tex.cli import LatexOCR
                _ocr_model = LatexOCR()
    return _ocr_model
```

### 🔴 缺陷 3：Ubuntu 环境 LibreOffice 中文字体缺失 — **确认真实问题**

**检索结论**：
- Ask Ubuntu 高票答案确认：`ttf-mscorefonts-installer` 需手动安装（含 EULA 交互确认）
- LibreOffice 命令行支持 `--infilter="Windows Metafile"` 处理 WMF 输入
- 中文字体（宋体/黑体等）在 Ubuntu 无头环境下也需额外安装

**修复**：在 Linux WMF 降级路径中增加日志警告；文档中注明 Ubuntu 部署需安装字体包。

---

### 🔴 缺陷 4：`str.replace` 占位符误伤风险 — **确认防御必要**

原方案用简单 `str.replace("[OCR_PENDING:rId123]", latex)` 替换。若正文碰巧出现相同字符串会导致误伤。

**修复**：使用 `re.sub` 精确匹配：
```python
re.sub(r'\[OCR_PENDING:' + re.escape(rid) + r'\]', latex, md_text)
```

## 2. 修正后的 `_clean_latex` 函数

```python
def _clean_latex(raw: str) -> str:
    """
    清洗 pix2tex 输出的 LaTeX：只剥离数学环境包裹标签，保留核心公式。
    """
    s = raw.strip()

    # 剥离 \begin{...} 和 \end{...} 环境标签（保留内部公式）
    for env in ('equation', 'equation*', 'align', 'align*', 'gather', 'gather*',
                'aligned', 'cases', 'array', 'matrix', 'pmatrix', 'bmatrix'):
        s = re.sub(r'\\begin\{' + env + r'\}', '', s)
        s = re.sub(r'\\end\{' + env + r'\}', '', s)

    # 剥离 \( \) 和 \[ \] 包裹
    s = re.sub(r'^\\\(\s*', '', s)
    s = re.sub(r'\\\)\s*$', '', s)
    s = re.sub(r'^\\\[\s*', '', s)
    s = re.sub(r'\\\]\s*$', '', s)

    # 剥离 $$ 和 $ 包裹（避免双包裹）
    s = re.sub(r'^\$\$\s*', '', s)
    s = re.sub(r'\$\$\s*$', '', s)
    if s.count('$') >= 2:
        s = re.sub(r'^\$\s*', '', s)
        s = re.sub(r'\$\s*$', '', s)

    return re.sub(r'\s+', ' ', s).strip()
```

## 3. 修正后的 OCR 单例 + 并发

```python
import threading

_ocr_model = None
_ocr_lock = threading.Lock()

def _get_ocr_model():
    """全局懒加载单例（线程安全双重检查锁定）"""
    global _ocr_model
    if _ocr_model is None:
        with _ocr_lock:
            if _ocr_model is None:
                from pix2tex.cli import LatexOCR
                import torch
                _ocr_model = LatexOCR()
                _ocr_model.model.eval()
    return _ocr_model

def _ocr_single(png_path: str) -> str:
    """单个公式 OCR（由线程池并发调用，共享全局模型）"""
    import torch
    from PIL import Image
    model = _get_ocr_model()
    img = Image.open(png_path).convert("RGB")
    with torch.no_grad():
        latex = model(img)
    return _clean_latex(latex)
```

```python
def wmf_to_png(wmf_path: str, png_path: str) -> bool:
    import platform

    # Tier 1: Pillow GDI (Windows)
    if platform.system() == "Windows":
        try:
            from PIL import Image
            img = Image.open(wmf_path)
            img.load()
            img.save(png_path, "PNG")
            return True
        except Exception:
            pass

    # Tier 2: Inkscape (跨平台)
    try:
        import subprocess
        subprocess.run(
            ["inkscape", wmf_path, "--export-type=png",
             f"--export-filename={png_path}"],
            capture_output=True, check=True, timeout=15
        )
        return True
    except Exception:
        pass

    # Tier 3: LibreOffice (Linux 无头)
    try:
        import subprocess, shutil
        out_dir = os.path.dirname(png_path)
        subprocess.run(
            ["soffice", "--headless", "--infilter=Windows Metafile",
             "--convert-to", "png", "--outdir", out_dir, wmf_path],
            capture_output=True, check=True, timeout=30
        )
        # LibreOffice 输出文件名基于输入名，需重命名
        base = os.path.splitext(os.path.basename(wmf_path))[0]
        expected = os.path.join(out_dir, base + ".png")
        if os.path.exists(expected) and expected != png_path:
            shutil.move(expected, png_path)
        return os.path.exists(png_path)
    except Exception:
        log.warning(
            "LibreOffice WMF 转换失败。Ubuntu 环境需安装: "
            "sudo apt install ttf-mscorefonts-installer fonts-noto-cjk"
        )
        return False
```
