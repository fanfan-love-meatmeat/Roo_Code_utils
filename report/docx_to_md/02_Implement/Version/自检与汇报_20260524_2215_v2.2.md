## 4. 移交

### 依赖安装

```bash
conda activate roo_mcp
pip install pix2tex torch
```

### 验证命令

```bash
python tools/docx_to_md.py "path/to/document.docx"
```

### MCP 使用

```json
{
  "convert_docx_to_md": {
    "docx_path": "D:/path/to/file.docx",
    "max_ocr_workers": 4
  }
}
```
