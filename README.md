# Roo_Code_utils MCP Server

专为学术科研与工程汇报优化的文档处理工具集。

## 🌟 功能特性
1. **convert_md_to_docx**: 支持中文宋体、英文 Times New Roman、蓝色表头及标准学术边距。
2. **extract_text_from_pdf**: 基于文本块提取，极大节省 LLM 处理 PDF 时的 Token 消耗。
3. **convert_pdf_to_images_fallback**: 仅在需要视觉确认复杂图表时使用的后备方案。
